Installation

วิธีติดตั้ง

เลือก plugin และวิธีติดตั้งที่ต้องการ

ติดตั้งผ่าน Claude Code Plugin Marketplace — Claude Code จะ clone repo และตั้งค่า MCP server อัตโนมัติ

1. เพิ่ม marketplace ใน settings.json:

// ~/.claude/settings.json { "extraKnownMarketplaces": { "medplansearch": { "source": { "source": "github", "repo": "winai363/MedPlanSearch" } } } }

2. ติดตั้ง plugin:

# ใน Claude Code terminal /marketplace add winai363/MedPlanSearch /plugin install medplansearch

หรือ ใช้ .mcp.json แบบ manual:

{ "mcpServers": { "MedPlanSearch": { "command": "node", "args": ["/path/to/MedPlanSearch/server.js"] } } }

ติดตั้งผ่าน npx หรือ npm — ไม่ต้อง clone repo เอง

วิธีที่ 1 — ผ่าน npx (แนะนำ):

// .mcp.json หรือ claude_desktop_config.json { "mcpServers": { "thai-quote-docs": { "command": "npx", "args": ["-y", "thai-quote-docs"] } } }

วิธีที่ 2 — ติดตั้ง global:

npm install -g thai-quote-docs
{ "mcpServers": { "thai-quote-docs": { "command": "thai-quote-docs" } } }

พร้อมเริ่มใช้งานแล้วหรือยัง?

ดู source code, ร่วมพัฒนา หรือรายงาน issue ได้ที่ GitHub
